Incoming Message Topic

Topic this device will subscribe to, and forward messages from.
NOTE: Router devices usually listen to the same topic sensor nodes send.
E.g.
Router: /beeline/in/#
Sensor: /beeline/out/#

Outgoing Message Topic

Topic this device will forward messages to.
NOTE: Router devices usually send to the same topic sensor nodes subscribe and listen.
E.g.
Router: /beeline/out
Sensor: /beeline/in

Message Routing

Idle Timeout

Seconds

How long to wait before disabling MQTT so it will never try to connect.
Used to save battery life.
0 = Keep MQTT connected even when its not being used. Device must be rebooted when this value is changed from 0.
