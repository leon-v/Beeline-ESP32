Display Configuration

Template


Text entered here will be displayed on the OLED display
To display message values, use this format:
[DEVLICE:SENSOR]
e.g. [:Battery] will show this devices battery voltage.

The device name can be omitted to show the value for this device.
e.ge.g. [Battery] will also show this devices battery voltage.

NOTE: Messages must be routed to this device's Display component before they show.
